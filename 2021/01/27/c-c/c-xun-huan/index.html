while语句

只要条件为真，while语句就能重复地执行循环体。
语法形式为：

while(condition){  //只要condition的值为真，就能一直执行
    statement
}

【注】:当不清楚要迭代多少次的时候，最好使用while循环。

do while语句

do while语句和while语句相似，唯一区别是do while语句先执行循环体后再进行检查条件。因此不管怎么样，do while都至少会执行一次循环。
语法形式为：

do
    statement
while(condition);

for语句

语法形式为：

for(init-statement; condition; expression)
    statement

for循环语句头中的init-statement,condition,expression可以都为空，但是分号必须保留。当这三者都为空时，程序将陷入死循环，但是可以通过break跳出。

break语句

可以出现在任意的循环体中，其作用是跳出循环。在多重循环的情况下，break只能跳出直接包含它的那一重循环。

#include <iostream>
using namespace std;
int main(){
    int n = 0;
    while (true){
        if (n > 100){
            break;
        }
        ++n;
    }
    cout << n; //打印结果：101
    return 0;
}

continue语句

终止最近的循环中的当前迭代并立即开始下一次迭代。（也就是立即终止本次循环，开始下一次循环）。
continue在多重循环里和break一致，continue只对自己包含它的那一重循环起作用。

#include <iostream>
using namespace std;
int main(){
    for (int i = 1; i <= 10; ++i) {
        if (i % 2)
            continue;
        cout << i << endl; //打印结果：2,4,6,8,10
    }
    return 0;
}

例：输入正整数n(n>=2),求不大于n的全部质数。

#include <iostream>
using namespace std;
int main(){
    int n;
    cin >> n;
    cout << 2 << endl;
    for (int i = 3; i <= n; i += 2) {
        int k;
        for (k = 3; k < i; k += 2) {
            if(i % k == 0){
                break;
            }
            if (k * k > i) {
                break;
            }
        }
        if(k*k > i){
            cout << i << endl;
        }
    }
    return 0;
}
